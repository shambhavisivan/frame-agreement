# Post-installation notes

New events and APIs have been added to FAM. Please consult the "FAM - API documentation" section of the User Guide to find out more.

Events:

- onBeforeCreateFrameAgreement
- onBeforeDeleteAddons
- onAfterDeleteAddons
- onAfterActivation

API:

- updateFrameAgreement
- createNewVersionOfFrameAgreement

New standard buttons are implemented for the Standalone Add-On feature:

- AddAddons
- DeleteAddons
- BulkNegotiateAddons

The above should be added to the *FA-Standard-Buttons* JSON Data.
